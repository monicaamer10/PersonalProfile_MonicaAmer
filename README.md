# Personal Profile — Monica Amer

This repository contains a static personal profile website. It can be published using GitHub Pages.

To publish:

1. Create a GitHub repository and push this code (see instructions below).
2. In the repository settings, enable GitHub Pages from the `main` branch (or `gh-pages` branch).

Or using the GitHub CLI (if installed and authenticated):

```bash
gh repo create <repo-name> --public --source=. --remote=origin --push
gh pages enable --branch=main
```

Then the site will be available at `https://<your-username>.github.io/<repo-name>/`.
